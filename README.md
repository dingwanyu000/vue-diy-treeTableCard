# vue-diy-treeTableCard
a diy treeTableCard of element-ui

基于Vue自定义树形tableCard

API
-----------------------
|Param          | Description   | Type  |
| ------------- |:-------------:| -----:|
| item          | 循环数据结构   | Array |
| noLine        | 是否有链接符   |Boolean|

|event          | Description   | param |
| ------------- |:-------------:| -----:|
|listenToChildPlus | 组件展开回掉事件   | |
|listenToChildMinus| 组件收缩回掉事件   | data |
